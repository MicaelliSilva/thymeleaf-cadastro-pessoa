#  Projeto Blog - Cadastro de Pessoas

## Descrição do Projeto
Este projeto é uma aplicação **Spring Boot com Thymeleaf** que realiza o **cadastro, listagem, edição e exclusão de pessoas**.  
O sistema utiliza uma interface web simples e intuitiva, permitindo gerenciar registros com campos como **nome, telefone, e-mail** e **data de nascimento**.

O objetivo é demonstrar o uso de **Spring MVC**, **Thymeleaf** e **JPA/Hibernate** com um banco de dados relacional, aplicando os conceitos de **CRUD** (Create, Read, Update, Delete).

---

## Tecnologias Utilizadas
- **Java 17+**
- **Spring Boot 3.x**
- **Thymeleaf**
- **Spring Data JPA**
- **Bootstrap 5**
- **Inputmask JS**
- **Banco de Dados H2 (em memória)**
